# Nextra Docs Starter 

This starter template includes the following features:

- [x] **Nextra 4**
- [x] **Search with Pagefind**
- [x] **Lucide Icons**

## Local Development

### Clone this repository

Using the GitHub CLI:
```bash
git clone https://github.com/AppointyTech/docs.git
```

### Install
```bash
bun i
```

### Run the development server
```bash
bun dev
```
